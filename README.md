# Rufus::Lua::Moon

Provides MoonScript for Rufus::Lua interpreter

## Installation

Add this line to your application's Gemfile:

    gem 'rufus-lua-moon'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rufus-lua-moon
